# N-Puzzle

This is an implementation of the classic 15-puzzle, 
generelized for N size ,  built with React, Redux and Bootstrap.

### Prerequisites

Node.js (version>=8)


### Installing
```
npm install
``` 

### Local Deployment
```
npm start 
```

### Packaging
```
npm run build
```
